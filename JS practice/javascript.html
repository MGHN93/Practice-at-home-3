Pair Programming Lesson 2: If/Else Conditions - Age Verification for Discounts
📌 Problem:

Create an age verification system for a store. Users enter their age, and the system tells them if they get a student discount (age 13-17), senior discount (age 65+), or regular price.

Real-world use: E-commerce sites use this for age restrictions, stores use this for discount calculations, websites use this for content filtering.

Key Concepts:
  • if/else if/else - Make decisions in your code
  • Comparison operators - ===, !=, <, >, <=, >=
  • Logical operators - && (AND), || (OR)
💻 Interactive Demo:
🔧 Your Turn - Write the Code:
// TODO: Complete this function

function checkDiscount() {
  // 1. Get age from input and convert to number
  // const age = parseInt(...);

  // 2. Validate age is a valid number
  // if (isNaN(age) || age < 0) { ... }

  // 3. Use if/else to check discount
  // if (age >= 13 && age <= 17) { ... }
  // else if (age >= 65) { ... }
  // else if (age < 13) { ... }
  // else { ... }
}
💡 Understanding if/else:
  • if (condition) { } - Run code if true
  • else if (condition) { } - Run if first is false, but this is true
  • else { } - Run if all previous are false
  • && - AND (both must be true)
  • || - OR (at least one must be true)
  • parseInt() - Convert string to number
  • isNaN() - Check if not a number
